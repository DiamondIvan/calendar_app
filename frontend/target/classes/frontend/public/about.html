Calendar & Scheduler App

Project Objective

Optimize Limited Daytime

Time management is essential. This app organizes time and commitments in a legible way.

Core Task: Build a Calendar and Scheduler App for personal use using Java (OOP, File I/O).

Constraint: No databases allowed. Must use local .csv files (event.csv, recurrent.csv).

Assignment Topic 2

PROJECT TEAM

Developing the Calendar App using Java and CSV storage:


Lead Developer

Logic & Recursion

Data Architect

File I/O & CSV Parsing

UI Designer

GUI & CLI Views

QA Engineer

Testing & Conflict Detection

Core Features

The application is designed to create and manage events, produce views for the user, and perform backup and restore functionality. Data persists in event.csv and recurrent.csv files.

Event Management

Create, Update, Delete

Auto-generated Event IDs

Start & End DateTime

Persistent storage in CSV

Search by date range

Advanced Logic

Recurring Events

Daily, Weekly, Monthly intervals

Repeat N times or until date

Conflict Detection:

Ensure no schedule clashes

System Utilities

Backup & Restore

Export all events to backup file

Restore to application CSVs
